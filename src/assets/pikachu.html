index.css единственный файл, в котором мы не используем css module
Во всех остальных компонентах, нужно использовать css module в проекте.
Также примени знания, которые ты узнал в уроках.
Не забудь сделать проверку, пришли ли props или нет.
Сделай три компонента <Header /> <Layout /> <Footer />, всех их размести в папке src/components а используй в компоненте <App />
Расположить их надо в следующем порядке:
<Header /> <Layout /> <Layout /> <Layout /> <Footer />

2.
Компонент <Layout /> имеет атрибуты title descr urlBg colorBg
3.
Компонент <Footer /> не принимает никаких props.
